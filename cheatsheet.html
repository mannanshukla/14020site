String Methods

there are many string methods which are similar to java

Get the character at position 1

a = "Hello, World!"
print(a[1])

this is substring, gets the characters

from postion 2 to postion 5

b = "Hello, World!"
print(b[2:5])

the strip() method will remove any whitespaces

from the beginning and end of a string

a = " Hello, World! "
print(a.strip()) # returns "Hello, World!"

the len() method will return the length of a string

a = "Hello, World!"
print(len(a))

the lower() method will return the string in lowercase

a = "Hello, World!"
print(a.lower())

the upper() method will return in uppercase

a = "Hello, World!"
print(a.upper())

the replace() method will replace the first arg by the 2nd arg

a = "Hello, World!"
print(a.replace("H", "J"))

note: this happens at all occurances

the split() method will split the string into substrings

if it finds instances of the seperator(the argument passed)

 a = "Hello, World!"
print(a.split(",")) # returns ['Hello', ' World!']

# Defines a "repeat" function that takes 2 arguments.
def repeat(s, exclaim):
    """
    Returns the string 's' repeated 3 times.
    If exclaim is true, add exclamation marks.
    """

    result = s + s + s # can also use "s * 3" which is faster (Why?)
    if exclaim:
        result = result + '!!!'
    return result
