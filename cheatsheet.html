Data Types

tuples are like lists, but cannot be changed later on

note: they are more lightweight than lists

you can make one like this:

langs= "python", "java", "perl"

The langs variable is of the type tuple

if you want to check the type of an object/variable in

python, you can use this:

type(programming_languages)

this will return the type, you can print this out

so running this:

langs= "python", "Java", "Perl"
print(type(langs))

will return

<class 'tuple'>

other data types are just like java

for example:

distro= "void"

will return:

functions

these are very very similar to java’s methods

# Defines a "repeat" function that takes 2 arguments.
def repeat(s, exclaim):
    """
    Returns the string 's' repeated 3 times.
    If exclaim is true, add exclamation marks.
    """

    result = s + s + s # can also use "s * 3" which is faster (Why?)
    if exclaim:
        result = result + '!!!'
    return result

here is the code that is being run:

def main():
    print repeat('Yay', False)      ## YayYayYay
    print repeat('Woo Hoo', True)   ## Woo HooWoo HooWoo Hoo!!!
