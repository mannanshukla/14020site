My Favorite Tech:

This is a collection of some of my favorite tech and what I use.

  • Laptop: HP EliteBook Folio 9470m

    • It is easy to upgrade, works well with Desktop Linux, had a pretty good keyboard and layout, includes a pointing stick, is in the sweet spot for cheap ram (DDR3) and has enough power for whatever I throw at it.

  • Operating System: Void Linux

    • Void Linux is an extremely minimal and fast distro. It uses runit as an init system rather than systemd this mitigates the one point of failure and make boot times quick. Its package manager is quite unique and void surprisingly supports flatpaks. Not to mention it's rolling release, so packages are usually quite recent.

  • Window Manager: i3-gaps

    • i3-gaps is my window manager, its tiling which means that floating windows are discouraged, this means no screen real estate is wasted. Gaps were added to make it easier to tell between different windows. It is highly configurable, fast and lightweight.

  • Text Editor: Vim

    • vim is just a meme, that's actually why I use it. Also its included in every linux repo known to man.

  • IEMs: Shure se315

    • The se315 are tuned to have a more balanced sound compared to the se215, they fit well in the ear and have very good noise-isolating tips. They were also quite the steal for the price I got them at.

  • Browser: Mozilla Firefox

    • The bloated web requires a bloated brower, Chrome is a botnet. That's actually why I use Firefox.

  • Storage Solution: 250GB Kingston a400

    • If you find an SSD for $30 you would buy it too, right?
